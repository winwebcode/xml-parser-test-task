Инструкция:

1. Настроить веб-сервер.
2. Создать БД xmlparser в MySQL.
3. В .env прописать доступ к БД.
4. Запустить миграции - php artisan migrate.
5. Перейти по адресу домена в директорию home, у меня https://xmlparser.loc/home
